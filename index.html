Психология травокура

(Формула успеха)

Психология травокура основана на том, что в некоторых странах наркотики декриминализованы, а в американских фильмах подростки курят «травку», как обычные сигареты. Молодые люди считают легкие наркотики безопасными и указывают, сколько живут травокуры на Западе. Это все правда!

    func toIntArray() -> [Int] {
        return self.map { char in

                fatalError()
            }
            return Int(char)
        }
    }
    func toStringArray() -> [String] {
        return self.map { String($0) }
    }
}
